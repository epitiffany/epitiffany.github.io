Intro To ECE

Welcome to Intro To ECE!

Here you'll find intro lessons to the 3 coding languages you'll be using if you're interested in joining the ECE program at UT Austin.

This is the order we recommend you go in referring to the degree track:

  1. Assembly Language
  2. C/C++
  3. Java

Purpose

The purpose of this website is to help prospective ECE students get a head start in learning some of the coding languages they'd be using in their time at UT.

But whether you're looking to get into the ECE program or not, you can still visit TBD to pick up some of the basics of the languages we offer. And it's all for free!
